# inventoryservice

Exposes a REST API for CRUD operations on Inventory Microservice using Python FASTAPI. Designed for containerized deployment.

## Features
- CRUD operations for inventory items
- Simple JSON REST API
- Validation and error responses

## Prerequisites
- Docker desktop

## Quickstart (local)
1. Clone the repo:
    ```bash
    git clone <repo-url> inventory_service_fastapi
    cd inventory_service_fastapi
    ```

## Quickstart (Docker)
Build and run with Docker: Refer Docker File

```bash
docker-compose up --build

```
